## 화면별 상세

### S-04: 홈 (메인)

**목적**: 재고 현황 한눈에 파악, 빠른 액션

**레이아웃**:
```
┌─────────────────────────────────────────┐
│ 🧊 우리집 냉장고          👨‍👩‍👧 가족명  │  ← Header
├─────────────────────────────────────────┤
│ ┌─────────────────────────────────────┐ │
│ │ 총 32개 품목 │ 유통기한 임박 5개   │ │  ← Summary Cards
│ └─────────────────────────────────────┘ │
├─────────────────────────────────────────┤
│ ⚠️ 유통기한 임박                        │  ← Section Title
│ ┌─────────┐ ┌─────────┐ ┌─────────┐    │
│ │ 🥛 우유  │ │ 🥬 상추 │ │ 🍖 삼겹살│    │  ← Expiring Items
│ │ D-1     │ │ D-2     │ │ D-3     │    │    (Horizontal Scroll)
│ └─────────┘ └─────────┘ └─────────┘    │
├─────────────────────────────────────────┤
│ 📦 전체 재고                      모두보기│  ← Section Title
│ ┌─────────────────────────────────────┐ │
│ │ 🥛 우유          1개      D-1 🔴   │ │
│ │ 🥬 상추          2봉      D-2 🟡   │ │  ← Stock List
│ │ 🍖 삼겹살        500g     D-3 🟢   │ │
│ │ 🥚 계란          10개     D-7 🟢   │ │
│ └─────────────────────────────────────┘ │
├─────────────────────────────────────────┤
│        ┌──────────────────┐             │
│        │  📷 영수증 스캔   │             │  ← FAB (Floating Action)
│        └──────────────────┘             │
├─────────────────────────────────────────┤
│  🏠 홈  │  🍳 레시피  │  📷 기록  │  ⚙️   │  ← Bottom Nav
└─────────────────────────────────────────┘
```

**컴포넌트**:
- `Header`: 앱 로고, 가족 이름
- `SummaryCard`: 총 품목 수, 임박 품목 수
- `ExpiringCarousel`: 유통기한 임박 품목 (가로 스크롤)
- `StockList`: 전체 재고 목록
- `ScanFAB`: 영수증 스캔 플로팅 버튼
