# 화면 구성

## 화면 목록

| ID | 화면명 | 설명 | 주요 컴포넌트 |
|----|--------|------|---------------|
| S-01 | 스플래시 | 앱 로딩 | Logo, Loading |
| S-02 | 온보딩 | 첫 사용자 안내 | Carousel, SkipButton |
| S-03 | 로그인 | 소셜 로그인 | SocialLoginButtons |
| S-04 | 홈 | 재고 현황 (메인) | Header, StockSummary, ExpiringList, QuickActions |
| S-05 | 재고 상세 | 품목별 상세 | ItemCard, EditForm |
| S-06 | 영수증 스캔 | OCR 스캔 | Camera, ImagePicker, ScanResult |
| S-07 | 스캔 결과 | OCR 결과 확인/수정 | ItemList, EditableRow |
| S-08 | 레시피 목록 | 추천 레시피 | RecipeCard, FilterTabs |
| S-09 | 레시피 상세 | 레시피 디테일 | RecipeHeader, Ingredients, Steps, StartCookingButton |

| S-12 | 기록 히스토리 | 식사 기록 캘린더 | Calendar, MealList |
| S-13 | 가족 관리 | 멤버 관리 | MemberList, InviteButton |
| S-14 | 설정 | 앱 설정 | SettingsList |

## 네비게이션 구조

│                                         │

## 화면별 상세

### S-04: 홈 (메인)

**목적**: 재고 현황 한눈에 파악, 빠른 액션

**레이아웃**:
```
┌─────────────────────────────────────────┐
│ 🧊 우리집 냉장고          👨‍👩‍👧 가족명  │  ← Header
├─────────────────────────────────────────┤
│ ┌─────────────────────────────────────┐ │
│ │ 총 32개 품목 │ 유통기한 임박 5개   │ │  ← Summary Cards
│ └─────────────────────────────────────┘ │
├─────────────────────────────────────────┤
│ ⚠️ 유통기한 임박                        │  ← Section Title
│ ┌─────────┐ ┌─────────┐ ┌─────────┐    │
│ │ 🥛 우유  │ │ 🥬 상추 │ │ 🍖 삼겹살│    │  ← Expiring Items
│ │ D-1     │ │ D-2     │ │ D-3     │    │    (Horizontal Scroll)
│ └─────────┘ └─────────┘ └─────────┘    │
├─────────────────────────────────────────┤
│ 📦 전체 재고                      모두보기│  ← Section Title
│ ┌─────────────────────────────────────┐ │
│ │ 🥛 우유          1개      D-1 🔴   │ │
│ │ 🥬 상추          2봉      D-2 🟡   │ │  ← Stock List
│ │ 🍖 삼겹살        500g     D-3 🟢   │ │
│ │ 🥚 계란          10개     D-7 🟢   │ │
│ └─────────────────────────────────────┘ │
├─────────────────────────────────────────┤
│        ┌──────────────────┐             │
│        │  📷 영수증 스캔   │             │  ← FAB (Floating Action)
│        └──────────────────┘             │
├─────────────────────────────────────────┤
│  🏠 홈  │  🍳 레시피  │  📷 기록  │  ⚙️   │  ← Bottom Nav
└─────────────────────────────────────────┘
```

**컴포넌트**:
- `Header`: 앱 로고, 가족 이름
- `SummaryCard`: 총 품목 수, 임박 품목 수
- `ExpiringCarousel`: 유통기한 임박 품목 (가로 스크롤)
- `StockList`: 전체 재고 목록
- `ScanFAB`: 영수증 스캔 플로팅 버튼
