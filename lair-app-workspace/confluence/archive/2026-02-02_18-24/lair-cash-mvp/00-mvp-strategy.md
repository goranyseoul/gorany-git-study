### 핵심 기능

| 기능 | 설명 | 상태 |
| --- | --- | --- |
| **지갑 생성** | Privy 이메일/소셜 로그인 → 자동 지갑 생성 | 🚧 개발 중 |
| **USDC 입금** | 이더리움 네트워크 USDC 입금 | 🚧 개발 중 |
| **금 축적** | USDC 예치 → XAUT(금) 이자 수령 | 🚧 개발 중 |
| **금 출금** | 만기 시 XAUT/USDC 출금 | 🚧 개발 중 |

### 수익화 (그로스)

| 기능 | 설명 | 상태 |
| --- | --- | --- |
| **배틀패스** | 예치/레퍼럴로 XP → 레벨업 → 금 보상 | 🚧 개발 중 |
| **금캐기 게임** | 탭하면 금 지급 (쿨타임 증가) | 🚧 개발 중 |
| **친구 초대** | 초대 보상 50 XP + 1mg | 🚧 개발 중 |
