## 1. 배경 및 목적

### 배경

한국의 많은 투자자들이 환율 하락 우려 속에 달러 자산에 관심을 갖고 있지만, 기존 금융 상품은 진입장벽이 높고, 크립토 금융은 복잡하다.

| 진입장벽 | 현재 상황 |
| --- | --- |
| 복잡한 용어 | DeFi, 스테이킹, 이자농사 등 암호화폐 전문용어 |
| 가스비 | ETH 등 가스 토큰을 별도 구매/출금해야 함 |
| 지갑 생성 | 시드구문 12개 백업 필요 |
| 신뢰 부족 | 거래소 해킹, 러그풀 등 불안 |

### 목적

**"퀴즈 풀고 금 받기"** — 달러와 금의 가치를 쉽게 배우고, 게임처럼 재미있게 금을 모으며, 자연스럽게 달러 예치로 전환시킨다.

### 핵심 가설

> 블록체인/금융에 익숙하지 않은 일반인에게 ① 퀴즈로 금융 지식을 알려주고 ② 게임처럼 금을 모으는 경험을 제공하면, 달러 예치까지 자연스럽게 전환할 수 있다.

### 핵심 가치 제안

| 구분 | 내용 |
| --- | --- |
| 쉬운 학습 | 10단계 퀴즈로 달러/금 투자 원리 학습 |
| 무료 체험 | 퀴즈 완주 시 금 10mg 지급, 금캐기 게임 무료 플레이 |
| 배틀패스 | 친구 초대 + 달러 입금으로 레벨업, 금 보상 획득 |
| 달러로 금캐기 | 달러 예치하면 자동으로 금을 획득 |

## 2. 타겟 유저

### 페르소나: 김달러 (32세, 직장인)

- 달러 투자에 관심 있지만 은행 예금 외에는 잘 모름
- 암호화폐는 복잡해서 시도해본 적 없음
- 게임은 좋아하고, 보상 받는 것에 민감함

### 유저 세그먼트

| 세그먼트 | 특징 | 목표 |
| --- | --- | --- |
| 호기심 유저 | 퀴즈에 관심 | 퀴즈 완주 → 가입 |
| 게이머 | 금캐기 게임에 관심 | 게임 플레이 → 예치 |
| 투자자 | 수익에 관심 | 달러 예치 → 리텐션 |

## 3. 기능 범위 (Scope)

### 포함 범위

| 분류 | 기능 |
| --- | --- |
| 온보딩 | 서비스 소개, 10단계 퀴즈, 소셜 로그인 |
| 계정 | 이메일/Google/Apple 로그인, 지갑 자동 생성 |
| 지갑 | USDC 수취 (Receive), 전송 (Send) |
| 가스비 대납 | Paymaster 크레딧 시스템 |
| 금캐기 게임 | 무료 게임, 60분 주기, 10% 랜덤 보상 |
| 친구 초대 & 배틀패스 | XP 기반 레벨/보상 시스템 |
| 친구 Nudge | 이탈 친구 콕 찔러 복귀 유도 |
| 푸시 알림 | 앱 외부 알림 (Nudge, 수확 등) |
| 전화번호 인증 | 배틀패스 보상 해금용 |
| 달러로 금캐기 | 달러 예치 → 금 보상 (Phase 1) |

### 제외 범위

| 기능 | 사유 |
| --- | --- |
| 멀티 체인 | MVP 복잡도 제한 |
| 토큰 스왑 | 규제 이슈 |
| 법정화폐 입출금 | 라이선스 필요 |
| 금 인출 | Phase 2 이후 |

## 5. 성공 지표 (KPI)

| 지표 | 목표 | 설명 |
| --- | --- | --- |
| 퀴즈 완주율 | 70%+ | 퀴즈 시작 → 완료 |
| 가입 전환율 | 80%+ | 퀴즈 완료 → 로그인 |
