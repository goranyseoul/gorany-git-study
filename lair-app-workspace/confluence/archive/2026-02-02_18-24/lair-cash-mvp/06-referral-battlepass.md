## 1. 개요

### 목적

- 친구 초대를 통한 바이럴 성장
- 배틀패스로 입금 및 참여 동기 부여
- 전화번호 인증으로 어뷰징 방지
- 친구 Nudge로 리텐션 강화

### 핵심 원칙

- **배틀패스 중심** — 레벨/보상 진행 상황을 명확히 시각화
- **초대 유도** — 친구 초대가 XP 획득의 핵심 수단
- **인증 필수** — 전화번호 인증 없이 배틀패스 보상 불가
- **소셜 연결** — 친구 Nudge로 이탈 유저 재유입

## 3. 배틀패스 카드 스펙

### 3.1 상단 요약

| 항목 | 표시 |
| --- | --- |
| 레벨 | "Lv.X" |
| 획득 보상 | "Xmg / 34mg" |
| XP 프로그레스 | "현재XP / 다음레벨XP" |
| 부스트 배지 | "⚡ 채굴량 5배 부스트" (활성 시) |

### 3.2 레벨 로드맵

- 1~10 레벨 노드 수평 표시
- 현재 레벨까지 색상 채워짐
- 각 노드 클릭 시 보상 확인 가능

---

## 4. XP 획득 방법

| 방법 | XP | 클릭 시 |
| --- | --- | --- |
| 친구 초대 | +50 XP / 1명 | 초대 링크 섹션으로 스크롤 |
| 달러 입금 | +5 XP / $1 | Receive 화면 이동 |
| 친구 Nudge 발송 | +1 XP / 1회 | 친구 목록으로 스크롤 |
| Nudge 성공 | +3 XP | 자동 지급 |

## 6. 내 친구들 섹션

### 6.1 친구 카드

| 항목 | 표시 |
| --- | --- |
| 닉네임 | 마스킹 (김**) |
| 관계 | "(내가 초대)" / "(나를 초대)" |
| 상태 | "채굴 중" / "수확 가능" / "대기 중" |
| Nudge 버튼 | 가능 시 활성화 |

### 6.2 Nudge 버튼 상태

| 친구 상태 | 버튼 |
| --- | --- |
| harvestable | [콕 찌르기] 활성화 |
| idle | [콕 찌르기] 활성화 |
| mining | "채굴 중" 텍스트 |
| 24시간 내 Nudge 발송 | "발송완료 ✓" 비활성화 |

## 7. 레퍼럴 보상 처리

### 7.1 새 친구 가입 시

1. 초대 링크로 앱 설치
2. 회원가입 완료
3. 초대한 사람에게 +50 XP 지급
4. 친구 목록에 추가

### 7.2 딥링크 처리

| 상황 | 처리 |
| --- | --- |
| 앱 설치됨 | 앱 열기 → 코드 자동 저장 |
| 앱 미설치 | 앱스토어 이동 → 설치 후 코드 저장 시도 |
| 딥링크 실패 | 가입 후 레퍼럴 코드 입력 화면 노출 |

## 8. 어뷰징 방지

### 8.1 전화번호 인증 기반 방지

| 정책 | 내용 |
| --- | --- |
| 1인 1번호 | 동일 전화번호로 중복 인증 불가 |
| 번호 변경 제한 | 인증 후 번호 변경 불가 (CS 문의 필요) |
| 가상번호 차단 | 070, 050 등 가상번호 인증 불가 |

### 8.2 추가 탐지 기준

| 유형 | 탐지 방법 |
| --- | --- |
| 다계정 생성 | 동일 기기 ID 체크 |
| 자기 초대 | 동일 IP + 기기 검증 |
| 봇 가입 | 행동 패턴 분석 |

### 8.3 제재 방법

| 단계 | 조치 |
| --- | --- |
| 1단계 | 경고 + 보상 취소 |
| 2단계 | XP 회수 |
| 3단계 | 계정 정지 |
