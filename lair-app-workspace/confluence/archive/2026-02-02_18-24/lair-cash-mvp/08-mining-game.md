## 1. 개요

탭 한 번으로 금을 채굴하는 게임. 탭할수록 쿨타임이 길어지며, Lv.5 달성 시 5배 부스트.

---

## 2. 유저 스토리

> **As a** 사용자  
> **I want to** 탭해서 금을 채굴하고  
> **So that** 재미있게 금을 모을 수 있다.

---

## 3. 핵심 메커니즘

### 3.1 채굴량

| 유저 등급 | 조건 | 채굴량/회 |
| --- | --- | --- |
| 일반 유저 | Lv.0~4 | 0.1mg |
| 부스트 유저 | Lv.5+ | 0.5mg |

### 3.2 쿨타임 (계속 증가)

| 탭 횟수 | 쿨타임 |
| --- | --- |
| 1회 | 없음 |
| 2회 | 1분 |
| 3회 | 5분 |
| 4회 | 30분 |
| 5회 | 1시간 |
| 6회 | 2시간 |
| 7회 | 4시간 |
| 8회 | 8시간 |
| 9회+ | 24시간 |

> ⚠️ 쿨타임은 리셋되지 않고 계속 증가

## 5. 게임 플로우

```
앱 실행 → 홈 → 금캐기 버튼 탭
→ (쿨타임 확인) → 채굴 가능이면 탭
→ 금 획득 애니메이션 → 잔액 업데이트
→ 다음 쿨타임 표시
```

---

## 6. API

| 엔드포인트 | 메소드 | 설명 |
| --- | --- | --- |
| `/mining/status` | GET | 쿨타임, 탭 횟수 조회 |
| `/mining/tap` | POST | 채굴 실행 |

### Response 예시

```json
{
  "success": true,
  "mined_amount": "0.1",
  "unit": "mg",
  "next_cooldown_seconds": 60,
  "tap_count": 2,
  "boost_active": false
}
```
