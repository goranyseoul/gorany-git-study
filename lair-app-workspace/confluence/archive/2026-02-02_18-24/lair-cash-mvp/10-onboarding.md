## 1. 개요

### 목적

- 로그인 없이 퀴즈를 통해 서비스 가치를 먼저 체험
- 10단계 퀴즈 완주 후 가입 유도 (지연된 가입)
- 금 보상으로 가입 동기 부여

### 핵심 원칙

- **무관문 진입** — 로그인 없이 바로 퀴즈 시작
- **쉬운 난이도** — 누구나 완주할 수 있도록 설계
- **보상 체감** — 퀴즈 완주 시 금 10mg 획득
- **스킵 옵션** — 귀찮은 유저를 위한 스킵 기능 (보상 없음)

---

## 2. 사용자 여정 (User Journey)

```
1. 서비스 소개 화면
   → 앱 첫 진입 시 서비스 핵심 가치 전달

2. 10단계 퀴즈
   → 로그인 없이 바로 시작
   → 쉬운 금융 퀴즈 10개를 풀며 학습
   → 정답 시 금 획득 축하 팝업 + 애니메이션
   → **스킵 가능** (보상 없이 바로 로그인으로 이동)

3. 완주 축하 화면
   → "금 10mg 발견!" 보상 인지
   → (스킵한 경우 이 화면 건너뜀)

4. 소셜 로그인
   → 이메일 / 구글 / 애플 로그인 선택
   → 금을 저장하기 위해 가입 유도

5. 레퍼럴 코드 입력
   → 레퍼럴 링크로 진입 시 자동 입력 후 스킵
   → 직접 진입 시 코드 입력 화면 표시

6. 홈 화면 이동
   → 가입 완료 후 홈 화면으로 이동
   → 퀴즈 보상 금 10mg 지급 완료 확인 (완주한 경우만)
```

## 3. 퀴즈 단계별 컨텐츠

| Step | 카테고리 | 질문 | 정답 |
| --- | --- | --- | --- |
| 1 | 달러 | 달러는 미국의 공식 화폐다 | O |
| 2 | 달러 | 달러는 전 세계에서 가장 많이 거래되는 화폐다 | O |
| 3 | 금 | 금은 역사적으로 안전자산으로 여겨진다 | O |
| 4 | 금 | 금 1g의 가격은 약 10만원이다 | X (약 13만원) |
| 5 | 투자 | 분산투자는 위험을 줄이는 전략이다 | O |
| 6 | 투자 | 달러와 금은 함께 오르는 경향이 있다 | X (역상관) |
| 7 | 스테이블 | 스테이블코인은 가격이 안정적인 암호화폐다 | O |
| 8 | 스테이블 | USDC는 달러와 1:1로 교환 가능하다 | O |
| 9 | 서비스 | Lair Cash에서 금을 모을 수 있다 | O |
| 10 | 서비스 | Lair Cash에서 친구를 초대하면 보상을 받는다 | O |

### 4.1 서비스 소개 화면

| 항목 | 스펙 |
| --- | --- |
| 헤드라인 | "퀴즈 풀고 금 받기" |

| 스킵 | 우측 상단 "건너뛰기" |

### 4.2 퀴즈 화면

| 항목 | 스펙 |
| --- | --- |
| 진행률 | 상단 프로그레스 바 (1/10) |
| 질문 | 중앙 카드 형태 |

| 정답 피드백 | 금 획득 애니메이션 + 팝업 |
| 오답 피드백 | "다시 생각해보세요" + 재선택 |
| 스킵 | 우측 상단 "건너뛰기" |

## 5. 에러 처리

| 상황 | 처리 |
| --- | --- |
| 네트워크 오류 | 재시도 버튼 표시 |
| 퀴즈 로딩 실패 | 기본 퀴즈 사용 (로컬 저장) |
| 보상 지급 실패 | 재시도 + CS 안내 |

## 6. 기존 유저 플로우

| 상황 | 동작 |
| --- | --- |
| 이미 가입된 유저 | 앱 진입 → 바로 홈 화면 |
| 퀴즈 중간 이탈 후 재진입 | 처음부터 다시 시작 |
| 퀴즈 완료 후 가입 안 하고 이탈 | 재진입 시 퀴즈 다시 진행 |
| 퀴즈 스킵 후 가입 안 하고 이탈 | 재진입 시 퀴즈 다시 진행 |
| 가입 완료한 유저가 퀴즈 재진행 원함 | 설정 > 튜토리얼에서 재진행 (보상 없음) |

## 7. 성공 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
| --- | --- | --- |
| 퀴즈 시작률 | 80%+ | 서비스 소개 → 퀴즈 시작 |
| 퀴즈 완주율 | 70%+ | 퀴즈 시작 → 10단계 완료 |
| 퀴즈 스킵률 | <20% | 퀴즈 시작 → 스킵 |
| 가입 전환율 | 80%+ | 퀴즈 완주 또는 스킵 → 로그인 완료 |
| 온보딩 완료율 | 45%+ | 앱 진입 → 홈 화면 도달 |
