## 1. 개요

사용자가 보유한 자산(USDC, XAUT)을 외부 지갑으로 전송하는 기능.

---

## 2. 유저 스토리

> **As a** 사용자  
> **I want to** 내 자산을 다른 지갑으로 전송해서  
> **So that** 자산을 자유롭게 이동시킬 수 있다.

---

## 3. 기능 상세

### 3.1 자산 선택

* USDC 또는 XAUT 선택
* 잔액 표시

### 3.2 받는 주소 입력

* 직접 입력 또는 QR 스캔
* 주소 유효성 검사 (0x로 시작, 42자)

### 3.3 금액 입력

* 숫자 키패드
* "전액" 버튼 (잔액 전체)
* 달러 환산 표시

### 3.4 전송 확인

* 받는 주소 (축약)
* 전송 금액
* 네트워크 수수료 (Paymaster 대납)
* "전송" 버튼

### 3.5 전송 결과

* 성공: 트랜잭션 해시, 잔액 업데이트
* 실패: 에러 메시지, 재시도 버튼

---

## 4. 화면 흐름

```
홈 → 자산 선택 → Send 버튼
→ 받는 주소 입력 → 금액 입력
→ 전송 확인 → 전송 중 (로딩) → 완료/실패
```

---

## 5. 주의사항

> ⚠️ **Ethereum 네트워크만 지원**  
> 전송 후 취소 불가

---

## 6. 기술 요구사항

| 항목 | 내용 |
| --- | --- |
| SDK | Privy Flutter SDK |
| 트랜잭션 | ERC-20 transfer |
| 가스비 | Paymaster 대납 |

---

## 7. API

| 기능 | 설명 |
| --- | --- |
| Privy SDK | `sendTransaction()` - 트랜잭션 전송 |
| Paymaster | 가스비 대납 처리 |

---

## 8. 에러 처리

| 에러 | 처리 |
| --- | --- |
| 잔액 부족 | "잔액이 부족합니다." |
| 잘못된 주소 | "올바른 지갑 주소를 입력하세요." |
| 트랜잭션 실패 | "전송에 실패했습니다. 다시 시도해주세요." |
| 네트워크 오류 | "네트워크 연결을 확인해주세요." |
