## 2. 파일럿 vs 정식 MVP 비교

| 항목 | 파일럿 | 정식 MVP |
| --- | --- | --- |
| **상품 구조** | 만기 있는 채권형 (여러 상품 동시 운영) | 만기 없는 무기한 상품 (단일 상품) |
| **이자율** | 고정 | 변동 (운용 수익에 따라) |
| **입금** | 모집 기간 중만 | 언제든 가능 |
| **출금 (원금)** | 만기 전 불가 | 언제든 가능 (대기 기간 있음) |
| **이자 출금** | 만기 전 불가 | 언제든 클레임 가능 |
| **컨트랙트** | 기존 재활용 | 전용 개발 (Audit 필요) |
| **이자 원천** | 마케팅 비용 | 예치금 운용 수익 |

## 3. 정식 MVP 상품 구조

> 만기 없이 언제든 입출금 가능한 **단일 무기한 상품**

| 항목 | 내용 |

| **입금** | 언제든 가능 |
| **출금** | 언제든 가능 (대기 기간 있음) |
| **이자 클레임** | 언제든 가능 |

## 4. 유저 플로우

### 입금 플로우

```
홈 화면 → 달러로 금캐기 카드 탭 → 금캐기 상세
→ 현재 이자율 확인 (변동)
→ 예치 금액 입력 → 예치 확인
```

### 출금 플로우

```
금캐기 상세 → 출금하기
→ 출금 금액 입력 → 출금 요청
→ 대기 기간 경과 → 출금 실행 (USDC 수령)
```

### 이자 클레임 플로우

```
금캐기 상세 → 적립된 이자 확인
→ 이자 클레임 → XAUT 수령
```

## 6. 기술 요구사항

### 스마트 컨트랙트 (전용 개발)

| 기능 | 설명 |
| --- | --- |
| **예치 (Deposit)** | USDC → Vault 예치 (언제든) |
| **출금 요청 (Request Withdraw)** | 출금 대기 기간 시작 |
| **출금 실행 (Withdraw)** | 대기 기간 후 USDC 출금 |
| **이자 클레임 (Claim)** | 적립된 XAUT 클레임 (언제든) |
| **이자 배분** | 하루 1회 자동 배분 |
