## 1. 개요

### 목적

- 달러/금 잔고를 한눈에 확인
- 금 캐기 게임 즉시 실행
- 예치 유도 및 배틀패스 홍보
- 핵심 기능으로의 빠른 접근

### 핵심 원칙

- **잔고 중심** — 보유 자산을 가장 먼저 표시
- **즉시 실행** — 금 캐기를 홈에서 바로 시작/수확
- **전환 유도** — 미예치 유저에게 예치 혜택 강조

## 3. 영역별 스펙

### 3.1 잔고 영역

| 항목 | 스펙 |
| --- | --- |
| 달러 잔고 | $X.XX (소수점 2자리) |
| 달러 표시 | "USDC" 라벨 |
| 금 잔고 | X.Xmg (소수점 1자리) |
| 받기 버튼 | → Receive 화면 이동 |
| 보내기 버튼 | → Send 화면 이동 |

### 3.2 금 캐기 게임 카드

**상태별 UI:**

| 상태 | CTA 버튼 | 타이머 |
| --- | --- | --- |
| idle | "금 캐기 시작" | - |
| mining | "채굴 중..." (비활성) | 남은 시간 표시 |
| harvestable | "수확하기" (애니메이션) | - |

### 3.3 배틀패스 미니 위젯

| 항목 | 스펙 |
| --- | --- |
| 레벨 | "Lv.X" |
| 보상 현황 | "획득mg / 최대mg" |
| XP 프로그레스 | 현재XP / 다음레벨XP |
| 클릭 시 | → 친구 초대 화면 이동 |

### 3.4 달러로 금캐기 카드

| 상태 | 표시 |
| --- | --- |
| 미예치 | "$10,000 예치 시 예상 / +15.3mg/일" |
| 예치 중 | "현재 획득 중 / +X.Xmg/일" |
| 상태 배지 | "미예치" / "예치 중" |

→ 클릭 시 달러로 금캐기 상세 페이지로 이동

**계산 기준:**

| 항목 | 값 |
| --- | --- |
| APR | 10% |
| 금 1mg | ₩260 |
| 환율 | $1 = ₩1,450 |

**예치 금액별 일일 금 획득량:**

| 예치 금액 | 일일 이자 | 일일 금 획득 |
| --- | --- | --- |
| $100 | $0.027 | 0.15mg |
| $1,000 | $0.27 | 1.5mg |
| $10,000 | $2.74 | 15.3mg |

### 3.5 하단 네비게이션

| 탭 | 아이콘 | 이동 화면 |
| --- | --- | --- |
| 홈 | 🏠 | 홈 화면 (현재) |
| 친구 초대 | 👥 | 친구 초대 & 배틀패스 |
| 설정 | ⚙️ | 설정 화면 |
