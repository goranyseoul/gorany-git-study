# 기능 정의서: 계정 (Account)

> **원본**: [Confluence](https://bugcity.atlassian.net/wiki/spaces/BUGCITY/pages/420708354)  
> **최종 수정**: 2026-01-12  
> **버전**: 0.3

---

## 1. 기능 개요

사용자가 앱에 가입하고 로그인하는 기능. 이메일 또는 소셜 로그인으로 인증하고, Privy를 통해 지갑이 자동 생성.

---

## 2. 핵심 플로우

```
앱 실행 → 스플래시 → 앱 소개 슬라이드 (3장) → 로그인 방식 선택 → 인증 → 약관 동의 → Privy 지갑 자동 생성 → 생체 인증 등록 → 초대 코드 입력 → 홈
```

---

## 3. 상세 스펙

### 3.1 로그인 방식

| 방식 | 지원 여부 | 비고 |
| --- | --- | --- |
| 이메일 | ✅ | 6자리 인증 코드 |
| Google | ✅ | OAuth 2.0 |
| Apple | ✅ | Sign in with Apple |
| Kakao | ❌ | Privy 미지원 |

### 3.2 앱 소개 슬라이드

| 항목 | 스펙 |
| --- | --- |
| 슬라이드 수 | 3장 |
| 네비게이션 | 스와이프 또는 [다음] 버튼 |
| 건너뛰기 | 우측 상단 "건너뛰기" 텍스트 버튼 |
| 노출 조건 | 최초 실행 시에만 (이후 자동 스킵) |

**슬라이드 구성:**

| 슬라이드 | 헤드라인 | 서브텍스트 |
| --- | --- | --- |
| 1 - 달러 지갑 | 내 폰에 달러를 보관하세요 | 은행 없이, 환전 수수료 없이 달러를 내 손 안에 보관 |
| 2 - 금 리워드 | 금을 모아보세요 | 퀴즈 풀고, 친구 초대하고, 금 보상을 받으세요 |
| 3 - 달러로 금캐기 | 달러로 금을 캐세요 | 달러를 예치하면 자동으로 금이 쌓여요 |

### 3.3 이메일 인증

| 항목 | 스펙 |
| --- | --- |
| 인증 코드 | 6자리 숫자 |
| 유효 시간 | 3분 |
| 재전송 쿨다운 | 60초 |
| 최대 시도 | 5회 실패 시 15분 대기 |

### 3.4 생체 인증

| 항목 | 스펙 |
| --- | --- |
| 지원 | Face ID, Touch ID (iOS) / 지문 (Android) |
| 필수 여부 | 선택 (스킵 가능) |
| 사용 시점 | 전송 시 확인 |
| 미지원 기기 | 자동 스킵 |

### 3.5 초대 코드

| 항목 | 스펙 |
| --- | --- |
| 딥링크 유입 시 | 자동 입력 → 화면 스킵 |
| 직접 진입 시 | 코드 입력 화면 표시 |
| 건너뛰기 | 가능 (나중에 입력 불가) |

## 4. 에러 처리

| 상황 | 처리 |
| --- | --- |
| 이메일 인증 코드 오류 | "인증 코드가 올바르지 않습니다" |
| 인증 코드 만료 | "인증 코드가 만료되었습니다. 다시 요청해주세요" |
| 로그인 취소 | 로그인 화면으로 복귀 |
| 소셜 로그인 실패 | "로그인에 실패했습니다. 다시 시도해주세요" |
| 지갑 생성 실패 | "지갑 생성에 실패했습니다. 다시 시도해주세요" |
| 생체 인증 실패 | 재시도 또는 스킵 옵션 |
| 초대 코드 오류 | "유효하지 않은 초대 코드입니다" |
| 네트워크 오류 | "네트워크 연결을 확인해주세요" + 재시도 버튼 |

## 5. QA 체크리스트

### 5.1 앱 소개 슬라이드

- [ ] 스와이프로 다음 슬라이드 이동
- [ ] [다음] 버튼으로 다음 슬라이드 이동
- [ ] [건너뛰기] 탭 시 로그인 화면으로 이동
- [ ] 3번째 슬라이드에서 [시작하기] 버튼 표시
- [ ] 인디케이터 현재 위치 정확히 표시
- [ ] 재방문 시 슬라이드 스킵 확인

### 5.2 이메일 인증

- [ ] 올바른 이메일로 인증 코드 수신
- [ ] 인증 코드 6자리 정상 입력
- [ ] 잘못된 코드 입력 시 에러 표시
- [ ] 3분 후 코드 만료 확인
- [ ] 60초 재전송 쿨다운 동작
- [ ] 5회 실패 후 15분 대기

### 5.3 소셜 로그인

- [ ] Google 로그인 성공
- [ ] Apple 로그인 성공
- [ ] 로그인 취소 시 정상 복귀
- [ ] 기존 계정으로 재로그인

### 5.4 생체 인증

- [ ] Face ID 등록 성공
- [ ] Touch ID/지문 등록 성공
- [ ] 스킵 후 정상 진행
- [ ] 미지원 기기에서 스킵 자동 처리

### 5.5 초대 코드

- [ ] 딥링크 유입 시 초대 코드 화면 스킵
- [ ] 유효한 코드 입력 시 정상 처리
- [ ] 유효하지 않은 코드 에러 표시
- [ ] 건너뛰기 후 정상 진행
