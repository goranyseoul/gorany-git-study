## 1. 개요

### 목적

* 무료로 즐길 수 있는 캐주얼 게임으로 신규 유저 유입
* 맛보기 경험 후 배틀패스 레벨업(입금/초대) 유도
* 끝까지 무료로만 이용하는 유저는 가치 없음 → 전환되지 않으면 버려도 됨

### 핵심 원칙

* **무료는 맛보기** — 2~3일 체험 후 전환 압박
* **쿨타임 영구 증가** — 리셋 없음, 계속 플레이할수록 불리
* **부스트는 Lv.5+만** — 입금/초대한 유저에게만 혜택
* **5배 차별화** — 확실한 부스트 체감

---

## 2. 유저 등급 및 채굴량

### 2.1 등급 정의

| 등급 | 조건 | 채굴량 | 원가 (1mg=₩260) |
| --- | --- | --- | --- |
| 무료 유저 | Lv.0~4 | 0.1mg | ₩26 |
| 부스트 유저 | Lv.5+ | 0.5mg | ₩130 |

### 2.2 Lv.5 기준 이유

* 배틀패스 Lv.5 = 누적 500 XP
* 달성 조건: 첫 $100 입금 OR 친구 10명 초대
* 서비스에 실질적 기여한 유저에게만 부스트 제공

---

## 3. 쿨타임 시스템

### 3.1 핵심 설계

* **리셋 없음** — 영구적으로 증가
* **완만한 증가** — 2~3일은 충분히 플레이 가능
* **60분 이후에도 5분씩만 증가** — 급격한 가속 없음

### 3.2 쿨타임 테이블

| 회차 | 쿨타임 | 누적 시간 |
| --- | --- | --- |
| 1 | 0분 | 0분 |
| 2~10 | 5분 | 45분 |
| 11~15 | 10분 | 1시간 35분 |
| 16~20 | 15분 | 2시간 50분 |
| 21~25 | 20분 | 4시간 30분 |
| 26~30 | 25분 | 6시간 35분 |
| 31+ | 30분 | +30분/회 |

## 4. 게임 플로우

```
1. 금 캐기 시작 (버튼 탭)
   → 채굴 애니메이션 시작
   → 쿨타임 타이머 시작

2. 쿨타임 대기
   → 화면에 남은 시간 표시
   → 푸시 알림 예약

3. 수확 가능
   → "수확하기" 버튼 활성화
   → 푸시 알림 발송

4. 수확
   → 금 지급 (0.1mg 또는 0.5mg)
   → 다음 회차 쿨타임 적용
   → 다시 1번으로

```

## 6. 배틀패스 연동

### 6.1 관계 정리

| 항목 | 금캐기 게임 | 배틀패스 |
| --- | --- | --- |
| 보상 형태 | 금 (mg) 직접 지급 | 금 (mg) 레벨 보상 |
| XP 획득 | ❌ 불가 | ✅ 입금/초대로만 |
| 부스트 조건 | Lv.5+ 필요 |  |

### 6.2 Lv.5 달성 방법 (배틀패스)

| 방법 | 필요량 | XP |
| --- | --- | --- |
| USDC 입금 | $100 | 500 XP → Lv.5 즉시 |
| 친구 초대 | 10명 | 500 XP → Lv.5 |

→ 금캐기만으로는 절대 Lv.5 불가  
→ 반드시 입금 또는 초대 필요
