## 1. Executive Summary

### 핵심 가설

> 블록체인 금융에 익숙하지 않은 일반 투자자들은 온체인 투자 서비스를 사용하면 더 다양한 금융 상품을 이용할 수 있지만 가스비와 체인 파편화 등 블록체인 서비스들이 가지는 고질적인 진입장벽 때문에 쉽게 접근하지 못하고 있다. 이 진입장벽을 해결해주면 스테이블코인 지갑 활성 유저를 확보할 수 있다.

### 최종 목표 지표

| 지표 | 설명 |
| --- | --- |
| **MAU** | 스테이블코인 지갑 월간 활성 유저 수 |
| **TVL** | 지갑에 입금된 스테이블코인 총량 |

### 핵심 가치 제안

**"블록체인 앱의 카카오페이"**

* 카카오페이가 온라인 결제 진입장벽을 없앤 것처럼
* Lair Cash가 블록체인 투자 서비스 진입장벽을 없앤다

---

## 2. 문제 정의: 블록체인 앱의 진입장벽

### 현재 유저가 겪는 Pain Points

| 진입장벽 | 기존 상황 | Lair Cash 해결책 |
| --- | --- | --- |
| **가스비** | ETH 또는 체인별 가스비 토큰 거래소에서 구매 후 개인지갑으로 스테이블 코인과 별도로 출금 필요함 | 가스비 대납 기능 제공 |
| **지갑 진입장벽** | 시드구문 12개 따로 백업해야 함 | Privy 임베디드 지갑으로 자동 생성 |
| **복잡한 개념** | DeFi, 스테이킹, 유동성 풀... 뭔지 모름 | 금캐기 컨셉으로 쉽게 설명 |
| **체인 파편화** | Base, Arbitrum, Polygon... 각각 가스비 필요 | 이더리움 단일 체인 |

## 3. MVP 목표

### 온보딩 Hook

> **게임화 + 학습 = 퀴즈 기반 온보딩**

| 구분 | 설명 |
| --- | --- |
| 목표 | 10단계 퀴즈로 달러/금 투자 원리 학습 |
| 보상 | 퀴즈 완주 시 금 10mg 지급 |
| 전환 | 금캐기 게임 → 배틀패스 → 입금 유도 |

### 바이럴 메커니즘

| 방법 | 보상 |
| --- | --- |
| 친구 초대 | +50 XP / 친구 1명 가입 |
| 달러 입금 | +5 XP / $1 |
| Nudge 발송 | +1 XP |
| Nudge 성공 | +3 XP |

## 4. MVP 개발 범위

### 기본 기능 (Basic)

* **계정:** 이메일 및 소셜 로그인 연동
* **지갑:** 지갑 생성, 스테이블 코인 수취(Receive) 및 전송(Send)
* **UX:** 가스비 대납(Paymaster) 적용 → 유저 경험 개선

### 부가 기능 (Phase별)

* **Phase 1:** 달러로 금 캐기 (이자 → 금 RWA 토큰)
* **Phase 2:** 달러 국채 투자
* **Phase 3:** 고래 포지션 알림
