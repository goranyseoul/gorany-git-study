## 1. 개요

사용자 계정 정보 확인 및 앱 설정을 관리하는 화면.

---

## 2. 화면 구성

```
┌─────────────────────────────┐
│  [←]        Settings        │
├─────────────────────────────┤
│  👤 계정 정보                │
│  ┌───────────────────────┐  │
│  │ user@email.com        │  │
│  │ 0x1234...abcd         │  │
│  └───────────────────────┘  │
│                             │
│  📞 전화번호 인증            │
│  ┌───────────────────────┐  │
│  │ [인증하기] 또는 ✅ 인증됨 │  │
│  └───────────────────────┘  │
│                             │
│  🔔 알림 설정                │
│  ┌───────────────────────┐  │
│  │ 푸시 알림     [Toggle] │  │
│  │ 마케팅 알림   [Toggle] │  │
│  └───────────────────────┘  │
│                             │
│  📄 약관 및 정책             │
│  ┌───────────────────────┐  │
│  │ 서비스 이용약관   [>]  │  │
│  │ 개인정보처리방침  [>]  │  │
│  └───────────────────────┘  │
│                             │
│  [로그아웃]                  │
│                             │
│  버전: 1.0.0                │
└─────────────────────────────┘
```

---

## 3. 메뉴 항목

### 3.1 계정 정보

| 항목 | 내용 |
| --- | --- |
| 이메일/소셜 | 로그인 계정 정보 |
| 지갑 주소 | 축약 표시 (탭 시 복사) |

### 3.2 전화번호 인증

| 상태 | 표시 |
| --- | --- |
| 미인증 | [인증하기] 버튼 |
| 인증됨 | ✅ 인증 완료 |

### 3.3 알림 설정

| 항목 | 설명 |
| --- | --- |
| 푸시 알림 | 전체 알림 On/Off |
| 마케팅 알림 | 마케팅 수신 동의 |

### 3.4 약관

| 항목 | 링크 |
| --- | --- |
| 서비스 이용약관 | 웹뷰 또는 외부 링크 |
| 개인정보처리방침 | 웹뷰 또는 외부 링크 |

### 3.5 로그아웃

| 동작 | 설명 |
| --- | --- |
| 로그아웃 | 확인 다이얼로그 → 세션 종료 → 로그인 화면 |

---

## 4. API

| 엔드포인트 | 메소드 | 설명 |
| --- | --- | --- |
| `/user/profile` | GET | 사용자 정보 조회 |
| `/user/settings` | PATCH | 알림 설정 변경 |
| Privy SDK | - | 로그아웃 |
