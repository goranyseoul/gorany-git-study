## 1. 개요

### 목적

* XP 기반 레벨 시스템으로 유저의 입금 및 친구 초대 유도
* 마케팅 예산 내에서 효율적인 보상 지급
* 레벨이 올라갈수록 XP당 보상 효율 감소 (체감 설계)

### 핵심 원칙

* **50 XP = 1mg 금 = ₩250** 기준 단가 (초반 레벨 기준)
* XP 획득 비율은 고정, 레벨업 요구량 증가로 체감
* 레벨이 올라갈수록 mg/XP 효율 감소

| $200 | 1,000 XP | Lv.7 | 16mg |

| 행동 | XP | 비고 |
| --- | --- | --- |

## 4. 보상 해금 (Claim)

### 4.1 해금 조건

| 조건 | 설명 |
| --- | --- |
| 레벨 도달 | 해당 레벨 XP 충족 |
| 전화번호 인증 | 1인 1계정 검증 완료 |

### 4.2 인터랙션

| 상태 | UI | 탭 동작 |
| --- | --- | --- |
| 미도달 | 잠금 아이콘 | 안내 툴팁 |
| 도달 + 미인증 | 노란 테두리 + "📱" | 전화번호 인증 화면 이동 |
| 클레임 가능 | 노란 테두리 + "🎁" | 클레임 바텀시트 |
| 클레임 완료 | 체크 아이콘 | 상세 정보 |

## 6. 핵심 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
| --- | --- | --- |
| 첫 입금 전환율 | 15%+ | 가입 → $100 입금 |
| 평균 초대 수 | 3명+ | 유저당 초대 친구 |
| Lv.5 달성률 | 20%+ | 전체 유저 중 |
| CAC | ₩1,000 이하 | 마케팅비/신규유저 |
| Nudge 성공률 | 30%+ | Nudge 발송 → 24시간 내 금캐기 |
