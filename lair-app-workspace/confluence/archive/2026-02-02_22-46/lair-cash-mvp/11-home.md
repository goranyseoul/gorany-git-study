## 1. 개요

### 목적

* 달러/금 잔고를 한눈에 확인
* 금 캐기 게임 즉시 실행
* 예치 유도 및 배틀패스 홍보
* 핵심 기능으로의 빠른 접근

### 핵심 원칙

* **잔고 중심** — 보유 자산을 가장 먼저 표시
* **즉시 실행** — 금 캐기를 홈에서 바로 시작/수확
* **전환 유도** — 미예치 유저에게 예치 혜택 강조

## 3. 영역별 스펙

### 3.1 잔고 영역

| 항목 | 스펙 |
| --- | --- |
| 달러 잔고 | USDC 잔액 + USD 환산 |
| 금 잔고 | mg 단위 |
| 받기 버튼 | Receive 화면 이동 |
| 보내기 버튼 | Send 화면 이동 |

### 3.2 금 캐기 게임 카드

| 상태 | 표시 | CTA |
| --- | --- | --- |
| 시작 가능 | "금 캐기 시작" | [⛏️ 금 캐기 시작] |
| 채굴 중 | "XX분 후 수확 가능" | 비활성화 |
| 수확 가능 | "수확 가능!" | [⛏️ 수확하기] (강조) |

### 3.3 부스트 배너

| 상태 | 표시 |
| --- | --- |
| Lv.5 미만 | "Lv.5 도달 시 5배 부스트 해금" + [배틀패스 보러가기] |
| Lv.5 이상 | "⚡ 5배 부스트 적용 중" |

### 3.4 달러로 금캐기 카드

| 상태 | 표시 |
| --- | --- |
| 미예치 | "$10,000 예치 시 예상 / +15.3mg/일" |
| 예치 중 | "현재 획득 중 / +X.Xmg/일" |

**계산 기준:**

| 항목 | 값 |
| --- | --- |
| APR | 10% |
| 금 1mg | ₩260 |
| 환율 | $1 = ₩1,450 |

### 3.5 하단 네비게이션

| 탭 | 아이콘 | 이동 화면 |
| --- | --- | --- |
| 홈 | 🏠 | 홈 화면 (현재) |
| 친구 초대 | 👥 | 친구 초대 & 배틀패스 |
| 설정 | ⚙️ | 설정 화면 |
