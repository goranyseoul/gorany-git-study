## 1. 개요

> **"달러 넣으면 금이 쌓인다"**

USDC 예치 → XAUT(금) 이자 수령. 파일럿 상품으로 고정 만기 구조.

---

## 2. 상품 구조

| 항목 | 내용 |
| --- | --- |
| 상품명 | [만기일] 만기 상품 |
| 이자율 | 연 12% (금 기준) |
| 만기 | 30일~60일 (상품별 상이) |
| 중도 출금 | 불가 |
| 이자 지급 | 만기 시 일괄 지급 |

---

## 3. 유저 플로우

### 3.1 예치 플로우

```
앱 실행 → 홈 → "달러로 금캐기" 카드 탭
→ 상품 목록 → 상품 선택
→ 예치 금액 입력 → 예치 확인
→ 트랜잭션 서명 → 예치 완료
→ 홈에서 예치 현황 확인
```

### 3.2 만기 플로우

```
만기일 도래 → 푸시 알림
→ 앱 실행 → 상품 상세
→ "출금하기" 버튼 → 출금 확인
→ 원금(USDC) + 이자(XAUT) 수령
```

## 5. 스마트 컨트랙트

| 항목 | 내용 |
| --- | --- |
| 컨트랙트 | ERC-4626 Vault (기존 재활용) |
| 예치 토큰 | USDC |
| 이자 토큰 | XAUT |
| 네트워크 | Ethereum |

### 주요 함수

| 함수 | 설명 |
| --- | --- |
| `deposit(uint256 amount)` | USDC 예치 |
| `withdraw()` | 만기 후 원금+이자 출금 |
